# **App Name**: Mercurio POS

## Core Features:

- Product List Management: Add products to the purchase list, calculating the total, tax, and discounts dynamically.
- Invoice Generation & Printing: Generate and display an invoice with all purchase details. Print the invoice using the built-in print function.
- Payment Processing: Enable partial payments and calculate the remaining balance due.
- Customer Management: Manage customer list and associate them with invoices, track payments, and overdue balances.
- Multi-Language Support: Toggle the language between Arabic (RTL) and French (LTR). The app switches its layout and terminology accordingly.
- Reporting and Analytics: Present visual reports using charts: Bar chart for product sales and pie chart for payment ratios by customer or product.

## Style Guidelines:

- Primary color: Teal (#008080) evoking trust and efficiency for financial interactions.
- Background color: Dark gray (#333333) provides a clean and neutral backdrop to ensure focus on the content.
- Accent color: Orange (#FFA500) to highlight important actions.
- Headline font: 'Poppins', a geometric sans-serif, for a modern, fashionable, precise feel.
- Body font: 'Poppins', a geometric sans-serif, for a modern, fashionable, precise feel. Note: currently only Google Fonts are supported.
- Use a set of minimalist icons for categories like product types and payment options.
- Employ CSS Grid and Flexbox for a fully responsive design that adapts to various screen sizes.
- Use subtle animations for transitions and feedback.
